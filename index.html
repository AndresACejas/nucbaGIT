Comandos para GIT

  1. Primer comando: git init (una sola vez por proyecto)
  2. Segundo comando: git status (para chekear el estado de git en nuestro proyecto
  3. Tercer comando: git add . (para trackear todos los archivos)
  4. Cuarto comando: git commit -m "mensaje" (para hacer el paquete de una version, que podemos subir a la nube)
  5. agregamos un cambio
